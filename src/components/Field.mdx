# Field

Super nice intro to the field component, what is doing and what its purpose is.

## Basic usage

<Playground>
    <Field label="E-mail" onSubmit={(stopEditing, submittedOn) => stopEditing()} renderEditor={(isEditing, stopEditing) => {
        const editor = isEditing ? (
            <input autoFocus={true} defaultValue="user@domain.tld" type="email" onBlur={stopEditing} />
        ) : <div className="value">user@domain.tld</div>;
        return (
            <div className={'editor' + isEditing ? ' active': ''}>
                {editor}
            </div>
        );
    }} />
</Playground>
